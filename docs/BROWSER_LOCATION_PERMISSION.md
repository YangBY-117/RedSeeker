# 浏览器位置权限设置指南

## 如何允许浏览器访问位置信息

### Chrome 浏览器

1. **方法一：通过地址栏图标**
   - 点击浏览器地址栏左侧的锁图标或信息图标
   - 找到"位置"选项
   - 选择"允许"
   - 刷新页面

2. **方法二：通过设置**
   - 点击右上角三个点菜单 → 设置
   - 进入"隐私和安全" → "网站设置"
   - 找到"位置信息"
   - 确保"询问前先询问（推荐）"已选中
   - 在"不允许访问位置信息的网站"中，删除你的网站（如果存在）
   - 刷新页面

3. **方法三：直接允许**
   - 当浏览器弹出位置权限请求时，点击"允许"
   - 如果之前拒绝了，需要清除网站数据后重新访问

### Edge 浏览器

1. 点击地址栏左侧的锁图标
2. 找到"位置"权限
3. 选择"允许"
4. 刷新页面

### Firefox 浏览器

1. 点击地址栏左侧的锁图标
2. 点击"更多信息"
3. 在"权限"标签中找到"访问您的位置"
4. 选择"允许"
5. 刷新页面

### Safari 浏览器

1. Safari → 偏好设置 → 网站
2. 选择"位置服务"
3. 找到你的网站，设置为"允许"

## 清除已拒绝的权限

如果之前拒绝了位置权限，需要清除：

### Chrome
1. 点击地址栏左侧的锁图标
2. 点击"网站设置"
3. 找到"位置"权限，改为"允许"
4. 或者点击"重置权限"

### 通用方法
1. 清除浏览器缓存和 Cookie
2. 重新访问网站
3. 当弹出权限请求时，选择"允许"

## 注意事项

- **HTTPS 要求**：现代浏览器要求使用 HTTPS 才能获取位置信息（localhost 除外）
- **首次访问**：首次访问网站时，浏览器会弹出权限请求
- **已拒绝**：如果之前拒绝了，需要手动在浏览器设置中允许

## 测试位置功能

1. 打开浏览器开发者工具（F12）
2. 查看控制台是否有位置相关的错误
3. 点击"当前位置"按钮
4. 如果浏览器弹出权限请求，选择"允许"
